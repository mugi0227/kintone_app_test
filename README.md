# キャリアアンカー診断 kintoneアプリ

## 📖 概要
このプロジェクトは、キャリアアンカー診断をkintone上で実施できるカスタマイズアプリです。

### 特徴
- 40問のアンケート形式（各1-5点）
- 回答後、最高点の問題から3つ選んで+4点のボーナス
- 8つのキャリアタイプ別に自動集計
- 最も高いタイプを自動判定

### 8つのキャリアタイプ
1. **専門・職能別コンピタンス (TF)** - 特定分野の専門性を追求
2. **全般管理コンピタンス (GM)** - 組織全体の管理・運営
3. **自律・独立 (AU)** - 自分のやり方で仕事をする
4. **保障・安定 (SE)** - 安定した雇用と収入
5. **起業家的創造性 (EC)** - 新しいビジネスを創造
6. **奉仕・社会貢献 (SV)** - 社会や他者への貢献
7. **純粋な挑戦 (CH)** - 困難な課題への挑戦
8. **生活様式 (LS)** - 仕事と私生活のバランス

## 🚀 セットアップ手順

### 1. kintoneアプリの作成
[docs/kintone-setup.md](docs/kintone-setup.md) を参照してアプリを作成してください。

### 2. JavaScriptカスタマイズの適用
1. kintoneアプリの設定画面を開く
2. 「JavaScript / CSSでカスタマイズ」をクリック
3. `src/career-anchor.js` をアップロード
4. `src/style.css` をアップロード
5. 「アプリを更新」をクリック

### 3. 動作確認
1. 新規レコードを作成
2. 40問のアンケートに回答
3. 自動的に「お気に入り選択」が表示される
4. 3つ選択して保存
5. キャリアタイプが自動判定される

## 💡 使い方
1. **診断を開始**: 新規レコードを作成
2. **質問に回答**: Q1〜Q40まで1-5点で回答
3. **お気に入り選択**: 最高点の問題から3つ選択（自動的に+4点）
4. **結果確認**: 8タイプのスコアと最終判定を確認

## 🔧 カスタマイズ
質問内容を変更したい場合は、`data/questions.json` を編集し、`src/career-anchor.js` の `QUESTIONS` 変数を更新してください。
